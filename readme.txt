1. В папке checkpoints содержатся:

	a) Предобученные веса мультиспикерного Tacotron2, частично совместимые с https://github.com/NVIDIA/tacotron2. Рекомендации по приведению к полной совместимости см. ниже.

	b) Предобученные на голосе Байдена веса вокодера HiFi в половинной точности (fp16), сконвертированные в статический torch граф. Для инференса предусмотрен метод infer. На вход вокодер принимает мел-спектрограммы, сгенерированные Tacotron2. Пример загрузки и использования такого графа:

	```
		import torch

		hifi = torch.jit.load("hifi_biden_fp16.jit")
		...
		
		audio = hifi.infer(taco_mel)
	```

2. Чтобы привести веса к полной совместимости с репозиторием NVIDIA, необходимо выполнить несколько простых действий:

	a) Добавить в text/symbols.py символ окончания строки _eos (например, _eos = "#") сразу после символа _pad, т.е. при кодировании текста символу _pad будет соответствовать индекс 0, а _eos - индекс 1;

	b) Проследить, чтобы итоговый вектор, полученный функцией text_to_sequence из скрипта text/__init__.py, содержал в конце индекс символа конца строки _eos;

	c) Отключить bias для свёрток, после которых идёт BatchNorm;

	d) Удалить из словаря весов Tacotron2 ключ 'speaker_embedding.weight', если вы не собираетесь модифицировать код и добавлять слой эмбеддинга дикторов.

3. В записях Байдена содержится много тишины как внутри записей, так и по их концам. Такая тишина может сильно влиять на сходимость механизма внимания, так что советуем провести программную очистку от тишины при помощи, например, librosa или sox.
